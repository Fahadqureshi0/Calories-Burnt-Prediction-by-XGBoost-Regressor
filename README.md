# 🔥 Calories Burnt Prediction using XGBoost Regressor

Predict the number of calories burned during exercise using Machine Learning. This project uses the **XGBoost Regressor** to estimate calories burned based on physiological and exercise-related features.

---

## 📌 Project Overview

Accurately estimating calories burned can help individuals monitor their fitness progress and manage their health. In this project, an XGBoost Regressor is trained on exercise data to predict calorie expenditure.

---

## 📂 Dataset

The dataset contains information about individuals and their exercise sessions.

### Features

- Gender
- Age
- Height
- Weight
- Duration
- Heart Rate
- Body Temperature

### Target

- Calories Burned

## 🤖 Model Used

### XGBoost Regressor

XGBoost is a powerful gradient boosting algorithm known for:

- High prediction accuracy
- Fast training
- Regularization to reduce overfitting
- Excellent performance on tabular datasets

## 📏 Evaluation Metrics

The model was evaluated using:

- R² Score
- Mean Absolute Error (MAE)

## 🚀 How to Run

Clone the repository

```bash

```

Move into the project directory

```bash

```

Install the required libraries

```bash

```

Run the notebook

